TinyDuckRun

개인정보 처리방침

TinyDuckRun은 오리 캐릭터를 조작해 장애물을 피하고 아이템을 모으는 모바일 게임입니다. 현재 버전은 회원가입, 광고, 분석 SDK 없이 동작합니다. 게임 기록은 기기에 저장되며, 이용자가 신기록 등록을 선택한 경우에만 닉네임과 점수가 공개 랭킹 저장소로 전송됩니다.

무료 앱 광고 없음 회원가입 없음 분석 SDK 없음 선택형 공개 랭킹
시행일 및 최종 업데이트: 2026년 4월 11일 앱 버전: 1.0.1

1. 처리하는 정보

TinyDuckRun은 게임 실행에 필요한 최소한의 기록만 처리합니다. 회원가입, 로그인, 결제, 광고, 행동 분석, 맞춤형 추적 기능은 제공하지 않습니다.

기기 안에 저장되는 내 기록 게임 일시와 점수가 앱 내부 저장소에 저장됩니다. 이 기록은 "내 기록 보기" 화면에서 표시되며, 개발자 서버로 자동 전송되지 않습니다.
선택적으로 등록하는 공개 신기록 이용자가 신기록 화면에서 닉네임을 입력해 등록할 때만 게임명, 닉네임, 점수, 등록 시간이 Supabase 기반 공개 랭킹 저장소에 저장됩니다.
서비스 운영 로그 공개 랭킹 등록 또는 조회를 위해 Supabase 서버에 접속할 때, 네트워크 보안과 오류 확인을 위한 요청 시간, IP 주소 등 기본 접속 정보가 인프라 로그로 처리될 수 있습니다.

2. 이용 목적

처리되는 정보는 아래 목적에만 사용됩니다.

  • 내 플레이 기록과 최고 점수 표시
  • 신기록 등록 시 공개 랭킹 저장 및 표시
  • 다른 사람 기록 보기 화면에서 점수순 또는 최신순 랭킹 제공
  • 랭킹 기능의 오류 확인과 악용 방지

3. 공개되는 정보

신기록 등록은 선택 사항입니다. 등록하면 입력한 닉네임과 점수, 등록 시간이 다른 이용자의 "다른 사람 기록 보기" 화면에 표시될 수 있습니다. 닉네임에는 실명, 전화번호, 이메일, 주소처럼 개인을 직접 식별할 수 있는 내용을 넣지 않는 것을 권장합니다.

4. 수집하지 않는 정보

TinyDuckRun은 아래 정보를 앱 기능을 위해 수집하거나 요청하지 않습니다.

  • 이름, 이메일 주소, 전화번호, 계정 비밀번호
  • 정확한 위치정보, 연락처, 캘린더, 사진 보관함
  • 카메라, 마이크, 음성 녹음
  • 광고 식별자, 맞춤형 광고 또는 앱 추적 목적의 정보
  • 결제 정보, 금융 정보

5. 기기 권한

현재 버전의 TinyDuckRun은 게임 플레이를 위해 카메라, 마이크, 위치, 연락처, 사진 보관함 같은 민감한 기기 권한을 요청하지 않습니다. 인터넷 연결은 공개 랭킹 등록과 조회에만 사용됩니다.

6. 제3자 처리

공개 랭킹 기능에는 Supabase가 사용됩니다. Supabase에는 신기록 등록 시 입력한 닉네임, 점수, 게임명, 등록 시간이 저장됩니다. 이 정보는 TinyDuckRun 랭킹 표시와 서비스 운영에 필요한 범위에서만 사용됩니다.

7. 보관 및 삭제

기기에 저장된 내 기록은 앱 내부 저장소에 보관됩니다. 앱을 삭제하거나 기기의 앱 데이터를 삭제하면 로컬 기록도 함께 삭제될 수 있습니다.

공개 랭킹에 등록한 닉네임과 점수의 삭제 또는 정정이 필요하면 아래 문의 경로로 요청해 주세요. 요청 내용을 확인한 뒤 합리적인 기간 안에 처리하겠습니다.

8. 아동의 개인정보

TinyDuckRun은 간단한 모바일 게임이며, 아동을 대상으로 개인정보를 고의로 수집하지 않습니다. 공개 랭킹 닉네임에는 아동의 실명, 연락처, 학교명 등 개인을 식별할 수 있는 정보를 입력하지 않도록 보호자가 안내해 주세요.

9. 변경 사항

향후 계정, 광고, 분석, 클라우드 저장, 푸시 알림 등 개인정보 처리와 관련된 기능이 추가되면, 기능 제공 전에 이 개인정보 처리방침을 업데이트하겠습니다.
